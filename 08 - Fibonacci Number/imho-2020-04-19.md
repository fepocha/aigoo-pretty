# for-loop

- Runtime: 48 ms, faster than 91.96% of JavaScript online submissions for Fibonacci Number.  
- Memory Usage: 33.8 MB, less than 100.00% of JavaScript online submissions for Fibonacci Number.

```js
var fib = function(N) {
  if (N === 0) return N;
  if (N === 1 || N === 2) {
    return 1;
  }

  var f1 = 1;
  var f2 = 1;
  var fn;

  for (let i = 2; i < N; i++) {
    fn = f1 + f2;
    

    f1 = f2;
    f2 = fn;

  } 

  return fn;
};
```
